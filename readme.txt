Инструкция по использованию бота:

1) в файле config.json - заполнить поля token(ввести в текстовое поле токен бота) и 
admins(ввести в массив через запятую id всех пользователей с правом администратор);
2) все файлы необходимо разместить на сервере, для запуска бота необходимо в корне проекта зпустить командами:
	$ npm install
	$ npm start
3) подключить бота к группе или каналу и сделать его администратором;
4) бот начинает работать сразу после подключения.

Команды бота:
	1) '/setting' - вызов меню настроек;
    2) '/statistic' - вся статистика за весь период работы;
    3) '/error' - узнать подробности об ошибках;
    4) '/del' - узнать подробности об удаленных сообщениях;
    5) '/unit_test' - провести тестирование бота на работоспособность.
